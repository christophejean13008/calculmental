Configuration Firebase

Pour synchroniser les données sur tous vos iPads, suivez ces 5 étapes. C'est gratuit et ne prend que 5 minutes.

📋 Instructions

1
Allez sur console.firebase.google.com et connectez-vous avec votre compte Google.
2
Cliquez « Créer un projet », donnez-lui un nom (ex: leitner-ecole), désactivez Google Analytics si proposé, puis « Créer ».
3
Dans le projet, cliquez sur l'icône Web </>, nommez l'app leitner, puis « Enregistrer ». Copiez l'objet firebaseConfig qui apparaît.
4
Dans le menu gauche, allez dans Build → Realtime Database, cliquez « Créer une base », choisissez mode test (30 jours). Ensuite dans l'onglet Règles, remplacez par :
{"rules":{".read":true,".write":true}} puis cliquez Publier.
5
Collez l'objet firebaseConfig ci-dessous et cliquez Enregistrer.
